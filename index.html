Gerador de Etiquetas

Aplicação desenvolvida em Java com Spring Boot para geração de etiquetas personalizadas em PDF. O sistema permite inserir nome e telefone e gerar automaticamente um arquivo pronto para impressão.

Tecnologias: Java, Spring Boot

Funcionalidades:

  • Cadastro de nome e telefone
  • Geração automática de etiquetas
  • Exportação em PDF
